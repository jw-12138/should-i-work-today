# v1 接口文档

包含额外的调试信息和随机描述文案。

## 查询今天

```http
GET /api/v1
```

## 查询明天

```http
GET /api/v1/tomorrow
```

## 响应格式

```json
{
  "requestTime": {
    "text": "2023/7/6 10:36:45",
    "timestamp": 1688611005000
  },
  "requestedDate": {
    "timestamp": 1688697405000,
    "day": "星期五",
    "text": "2023/7/7 10:36:45"
  },
  "todayIs": "weekday",
  "shouldIWorkToday": true,
  "desc": "好累啊"
}
```

## 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `requestTime.text` | string | 请求时间（格式化） |
| `requestTime.timestamp` | number | 请求时间戳（毫秒） |
| `requestedDate.timestamp` | number | 查询日期的时间戳（毫秒） |
| `requestedDate.day` | string | 星期几 |
| `requestedDate.text` | string | 格式化的日期时间 |
| `todayIs` | string | 日期类型 |
| `shouldIWorkToday` | boolean | 是否为工作日 |
| `desc` | string | 随机描述文案 |

## todayIs 取值

| 值 | 说明 |
|------|------|
| `weekday` | 普通工作日 |
| `weekend` | 普通周末 |
| `off_day` | 法定节假日 |
| `not_off_day` | 调休补班日 |
